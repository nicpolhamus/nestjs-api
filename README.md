# [Nestjs](https://docs.nestjs.com/) API with [Sequelize](https://sequelize.org/)

## Description

[Nest](https://github.com/nestjs/nest) api that uses sequelize for basic CRUD endpoints to manipulate user objects.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
